# ExploreCity - Қалалық мәдени іс-шаралар порталы

Жоба КМ04 пәні бойынша «Web-сайтты жобалау және үздіксіз жұмыс істеуін қамтамасыз ету» курстық жұмысына арналып жасалған. Бұл толыққанды **Full-stack (React + Node.js + SQLite)** веб-қосымшасы. Сайт Қазақстанның мәдени іс-шараларына арналған.

## 🚀 Қолданылған технологиялар (100 ұпай критерийлері бойынша)

### Фронтенд (React.js)
- **Фреймворк**: React + Vite (жылдам жүктелу).
- **Роутинг**: `react-router-dom` (Сайт ішіндегі логикалық навигация).
- **Мультиязычность**: `react-i18next` (Қазақ, Орыс, Ағылшын тілдері).
- **Дизайн**: Premium UI Glassmorphism (Қаріптер, түстер, иерархия сақталған, толық бейімделген / адаптивті).
- **Код сапасы**: Функционалды компоненттер, қателерді өңдеу, Props/State басқару.

### Бэкенд (Node.js & Express)
- **API**: Толық REST API (GET, POST, PUT, DELETE). 
- **Аутентификация**: JWT (JSON Web Tokens) және парольдерді `bcrypt` арқылы хештеу.
- **Деректер қоры (База данных)**: `PostgreSQL`. 
> ⚠️ **Маңызды**: `pgAdmin` ішінде `explorecity` атты жаңа мәліметтер базасын (Database) құру қажет! 
Сайттың `server/db.js` ішінде пароль ретінде `admin` көрсетілген (егер сіздің PostgreSQL пароліңіз басқа болса, сол файлға кіріп өзгертіңіз).
- **Қателерді өңдеу**: Барлық бөліктер `try-catch` арқылы қорғалған, HTTP қателер дұрыс қайтарылады.

## 📋 Жоба бойынша нұсқаулық (Қалай іске қосу керек?)

### 1-Қадам: Бэкенд серверін қосу және Серверді баптау
Ең алдымен компьютеріңізде **PostgreSQL** қосылып тұрғанына және `explorecity` базасы `pgAdmin` арқылы құрылғанына көз жеткізіңіз. Сосын:
Терминалды (командалық жолды) ашып, `server` папкасына өтіңіз:
```bash
cd server
npm install
npm run dev
```
> Бэкенд `http://localhost:5000` портында іске қосылады және барлық кестелер базада автоматты түрде құрылады.

### 2-Қадам: Фронтендті қосу
Екінші терминалды ашып, `client` папкасына өтіңіз:
```bash
cd client
npm install
npm run dev
```
> Фронтенд `http://localhost:3000` (немесе 5173) портында іске қосылады.

## 🔗 API Документация (API Documentation)

### 1. Авторизация
- **POST** `/api/auth/register` - Жаңа пайдаланушыны тіркеу (`name`, `email`, `password` жіберу керек).
- **POST** `/api/auth/login` - Жүйеге кіру (`email`, `password` жіберіп, JWT токен алады).

### 2. Іс-шаралар (Мероприятия)
- **GET** `/api/events` - Барлық іс-шараларды алу (Қала бойынша фильтрация бар: `?city=Almaty`).
- **GET** `/api/events/:id` - Нақты бір іс-шараны алу.
- **POST** `/api/events` - Шара қосу (Авторизация қажет, `Bearer token`).
- **PUT** `/api/events/:id` - Шараны өңдеу (Тек автор немесе админ).
- **DELETE** `/api/events/:id` - Шараны өшіру (Тек автор немесе админ).
